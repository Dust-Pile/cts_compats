CTS Compats (for short) is a flexible compatibility addon for Countered's Terrain Slabs. I have worked closely with countered in order to allow forge mod compatibility with the fabric based mod.

This mod is in early development! I hope to have a better version out soon... running into some mixin problems ;-;

Compatibility available for any of the following mods (hoping to add more soon once the workflow is solid):
- Biomes O' Plenty
- Project Vibrant Journeys



Also, additional vanilla features are available such as dripstone slabs and sweet berries on top.
